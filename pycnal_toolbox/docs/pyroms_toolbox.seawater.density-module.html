dens(S, T, P=0)

source code  
Compute density of seawater from salinity, temperature, and pressure

Usage: dens(S, T, [P])

Input:               
    S = Salinity,     [PSS-78]
    T = Temperature,  [°C]
    P = Pressure,     [dbar = 10**4 Pa]
P is optional, with default value zero

Output:
    Density,          [kg/m**3]

Algorithm: UNESCO 1983
